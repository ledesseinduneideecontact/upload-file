# Importer Fichiers - Webapp QR Code

Application web permettant d'importer des fichiers (photos/vidéos) via QR code depuis un mobile.

## Installation

```bash
npm install
```

## Lancement

```bash
npm start
```

Ouvrez votre navigateur sur `http://localhost:3000`

## Fonctionnalités

- ✅ Génération de QR code pour chaque session
- ✅ Upload de fichiers (photos/vidéos) depuis mobile
- ✅ Mise à jour en temps réel via Socket.IO
- ✅ Téléchargement individuel ou collectif
- ✅ Suppression de fichiers
- ✅ Interface responsive
